# Fusentodo（付箋アプリ） 使い方マニュアル

**「インストール不要でインターネットなしでも動くブラウザアプリです」**

## 基本的な使い方
1. このフォルダに入っている `index.html` をダブルクリックして、お手持ちのブラウザ（EdgeやChromeなど）で開きます。
2. 画面右下にある「＋」アイコンをクリックすると、付箋ボードが右から引き出されます。
3. テキスト入力欄にメモを入力し、「Enter」キー（もしくは「追加」ボタン）を押すと新しい付箋が追加されます。
4. 不要になった付箋は、それぞれの付箋内にある「ゴミ箱アイコン」をクリックすることでサッと消去できます。

## 別のパソコンや環境への移行について

### アプリケーション自体の移行（◎ とても簡単です）
このアプリは開発ツールなどのインストール作業や、難しい環境構築が一切不要です。
現在お使いの `fusentodo` フォルダ（中にある index.html, style.css, script.js の3つのファイル）を、そのままUSBメモリやOneDriveなどに入れて別のPCにコピーするだけで、別のパソコンでも同じように使うことができます。

**【パス（PATH）などの難しい設定は一切不要です！】**
- 古いシステムなどで気にする必要があった「環境変数の設定」や「パスを通す」といった面倒な作業は**完全に不要**です。
- ファイル同士が「同じフォルダの中にいること」でプログラムが連携する仕組み（相対パス）になっているため、3つのファイルが1つのフォルダ内にまとまってさえいれば、USBの中、Dドライブ、デスクトップなど、どこに配置しても `index.html` をダブルクリックするだけで確実に動作します。

### 【重要】「書き込んだ付箋データ」の移行（△ そのままでは移動しません）
付箋に書いた文字データは、HTMLファイルの中ではなく「今使っているパソコンの、Edgeブラウザの内部（ローカルストレージ）」に保存されています。

そのため、以下の点にご注意ください。
- アプリのフォルダを別のPCにコピーして開いた場合、新しいPCでは**「空っぽの初期状態」**として開かれます。（古いPCのデータは古いPCのEdgeの中に残ったままです）
- 万が一ブラウザの「閲覧データをすべて消去する」機能を実行した場合、これまでのメモも消えてしまいます。

**※データも一緒に移動できるようにしたい場合**
もし「今のPCで書いたメモを、別のPCでも引き継ぎたい」「バックアップを取れるようにしたい」という場合は、「データを書き出して保存（エクスポート）」機能と「データを読み込む（インポート）」機能のボタンを追加することで解決できます。（いつでも追加可能です）
